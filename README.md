# Theme Changer — useContext API

A simple React project built to learn and implement the **Context API**.

## What I practised

- Creating context using `createContext()`
- Wrapping components inside a `Provider`
- Consuming context in child components using `useContext()`
- Managing multiple contexts in the same app (`UserContext` and `ThemeContext`)

## Tech Stack

React · Vite · CSS

## Run locally

```bash
npm install
npm run dev
```
